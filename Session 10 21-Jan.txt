HTTP Status code:
------------------
http://amaravathi.today/ai/transactions

100 -> Informational
continuing or switching protocols from HTTPS to HTTP

2XXX - -> Success
	200 - Ok
	201 - created
	204 - Deleted

3XXX --> Redirectional
	301 - url redirection perm
	302 - temp redirection
	304 - Browser cache

4XXX - Client side error
	404 - Not found
	400 - Bad request, Invalid Json
	401 - Unauthorize, No authentication
	403 - Forbidden, No peermission to this

5xxx - Server side error
	500 - Internal server error
	502 - Bad gateway
	503 - Service unavailable
	504 - Gateway timeout
